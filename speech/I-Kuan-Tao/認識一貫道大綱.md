主題：認識一貫道

一、從生命安頓切入道的意義

（一）提出切身問題

1. 一貫道不只是宗教知識
2. 人一生都在尋找心的安頓
3. 心有方向，境界中較不亂

（二）回到本心認識道

1. 道是貫通天地的天理
2. 道也是人人本具的良心善性
3. 會慚愧、願向善，就是道心發露

二、說明一貫的根本精神

（一）以聖賢教化印證

1. 孔子說「吾道一以貫之」
2. 佛家講佛性，儒家講良知
3. 道家講萬物得一而生

（二）收回共同本源

1. 名稱不同，根本同歸本心
2. 一貫不是比較高下
3. 各教都提醒人不要迷失自己

三、建立一貫道的傳承脈絡

（一）承接聖聖相傳心法

1. 陸象山說人心具此理
2. 真道不受時代地域限制
3. 聖賢所救的是人的迷心

（二）簡述近代歷史開展

1. 黃德輝祖師開創先天道
2. 王覺一祖師闡揚真儒心性
3. 劉清虛祖師改稱一貫道
4. 師尊師母大開普渡收圓

四、看見先得後修的慈悲

（一）說明普渡精神

1. 過去求道條件嚴格
2. 白陽道運重在庶民普渡
3. 先給人回頭機會，再慢慢修正

（二）以夏盛珍老前人為例

1. 沉迷鴉片，仍有善根
2. 求道後立志戒除習氣
3. 清口開荒，成全多人
4. 道看重願不願意回頭

五、把一貫道落實在生活中

（一）宗旨落在人倫日用

1. 知恩感恩：敬天地、孝父母
2. 修身立品：改脾氣、去毛病
3. 人倫實踐：信朋友、和鄉鄰
4. 立己立人：己立立人，推向大同

（二）禮節與三寶提醒修道

1. 獻香叩首幫助收回散亂心
2. 禮節是感恩，不只是形式
3. 三寶指出生命歸根路
4. 得道是開始，修道才是實行

六、從自己開始承擔使命

（一）修道就在眼前

1. 真實考場在家庭與人事
2. 境界照出自己的習氣
3. 行功了愿從一念愿心開始

（二）道傳世界從我做起

1. 前賢以生命願力傳道
2. 別人常從我們的態度認識道
3. 世界大同從當下一念修心開始

七、結語回到原來的自己

（一）真正認識一貫道

1. 不只知道歷史、宗旨、禮節
2. 要認出本來清淨的良心善性
3. 讓道理在自己的心裡醒過來

（二）收束勉勵

1. 從良心、感恩、改過出發
2. 把道理活在日常生活
3. 在生命裡真正走上一貫道
