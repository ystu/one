# 講道知識庫

這個 repo 用來整理講道相關的題材、經典、故事、提示詞與講稿素材。

## 目的
- 把零散靈感變成可查找的知識庫。
- 把題材與講稿成品分開。
- 讓之後寫講稿、備課、延伸系列主題時，不必每次從零開始。

## 原則
- `docs/` 放知識與工具。
- `speech/` 放講稿與系列資料。
- 題材先拆小、先分類，再慢慢長成完整內容。
